# Product Vision: Website Intelligence Platform (WIP)

## Mission
Observe, understand, reconstruct, and validate any web experience using AI and browser instrumentation.

## Core Philosophy
**Separate observation from reasoning.**

Never allow the LLM to guess something the browser can measure.

## The Long-Term Vision
The most ambitious version of this project is not a "website cloning AI." It is a **Web Intelligence Operating System**.

Its responsibilities are to:
1. Execute web applications in a real browser.
2. Observe every aspect of their visual, structural, semantic, and interactive behavior.
3. Build a persistent, queryable Observation Graph that captures the application's experience over time.
4. Allow AI to reason over that graph instead of raw pixels or source code.
5. Validate reconstructed models against the live application.
6. Serve as a foundation for products such as Forge, UI migration tools, accessibility auditing, design system extraction, automated documentation, and regression testing.

## Consumers
- **Forge** becomes Consumer #1 of WIP.
- Other future consumers: React Adapters, Flutter Adapters, HTML Adapters, Accessibility Analyzers, UI Test Generators, Design Token Extractors, UX Analytics, Documentation Generators.
